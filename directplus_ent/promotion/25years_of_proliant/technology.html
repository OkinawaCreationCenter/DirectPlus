  • 「ProLiant」の独創のテクノロジー

独創のテクノロジーがx86サーバーの価値を高める。

ProLiantは、業界標準のx86サーバーでありながら、管理性や信頼性を高める独創的なテクノロジーを数多く採用しています。サーバーのライフサイクル全体を通していかに運用管理性を高めるか、日々の運用の中でより高い性能と信頼性をいかに両立させるか――ProLiantは、サーバー運用の現場を知るお客様 の声を反映させながら進化し続けています。

最先端の管理プロセッサー

HPE Integrated Lights-Out 5 (iLO 5)

遠隔操作やサーバー運用の「自働化」機能に加え、世界に先駆けてシリコンレベルのセキュリティを実装したHPE独自開発の管理プロセッサーです。

HPE Smartメモリ

HPE Memory Fast Fault Toleranceと呼ばれるメモリRAS機能を実装し、耐障害性を向上させながら優れたパフォーマンスを発揮します。

進化する内蔵ストレージ

HPE Smartアレイ

1枚のアレイコントローラーでRAIDモードとHBAモードの切り替えに対応します。また、ノード単位のライセンスによりコストを抑えながら暗号化が可能です。

詳細はこちら

HPE Smartキャッシュ

独自のアルゴリズムによりデータのアクセス傾向を自動的に分析。
アクセス頻度の高いデータを高速なSSDにキャッシュして内蔵ストレージを高速化します。

HPE Smartストレージバッテリ

バックアップ電源を提供し、何らかの理由で電源が失われたときに、キャッシュに保存されたデータをフラッシュメモリへ転送することでデータを保護します。

HPE Smartドライブキャリア

ホットプラグ対応HDDに視認性に優れたステータスLEDを採用し、誤認や誤操作によるデータ消失のリスクを低減。「取り出し禁止」を示す表示も一目瞭然です。

※各表示機能はHP Smartアレイコントローラーとの接続時にのみ動作し、オンボードSATAコントローラー等ではサポートされません。

一歩先を行く管理性

iLOサービスポート

専用のUSBポートにPCを接続してiLO 5やリモートコンソール等に直接アクセス可能。データセンターやマシンルームでの管理を容易にします。

REST API

オープンなREST APIに対応し様々なソフトウェアやツールとの連携を容易にしています。サーバー管理をよりシンプルかつ高度に進化させます。
