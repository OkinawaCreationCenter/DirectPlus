激動する時代の中で、時代が求める役割を担い続けた。

世界初のx86サーバー「ProLiant」が採用したOSは、Windows NT Advanced Server version 31でした。
そして、オープン化とダウンサイジング、インターネットとネットワークコンピューティング、クラウドコンピューティングと大きく変わる時代とともに、「ProLiant」の担う領域は大きな広がりを見せました。
いま、ProLiantはハイブリッドITの時代においてさらに活躍の場を拡大しています。

オープン化の潮流、x86サーバーの登場

1990年代に始まったメインフレームからUNIXシステムへのダウンサイジングは、1993年に登場したx86サーバー「ProLiant」による真のオープン化とともに加速度的に進展していきました。

サーバーOSと仮想化ソフトウェア

Windows NTからWindows Serverへ

x86サーバー「ProLiant」とWindows Server OSは、オープンシステムの両輪として新しい市場を切り拓いていきました。そして、ProLiant はインターネット時代の主役としても活躍していったのです。

サーバー仮想化テクノロジーの進展

2000年代に入って、x86サーバー向けのハイパーバイザー上で複数の仮想マシン（VM）を稼働させる環境が一般化。ProLiantは、仮想化によるサーバー統合プラットフォームとして市場をリードしました。

“自働化”を身に着けたx86サーバー

2012年に登場した“自働サーバー”は、導入・監視・運用・保守に至るライフサイクル全体の自動化を推し進めました。ProLiant Gen8の登場が、その後のサーバーの進化の方向性を決定づけたと言えるでしょう。

サーバー管理

増大するサイバー攻撃の脅威に対抗

サイバー攻撃の脅威は、アプリケーションやOSだけでなくサーバー機器そのものにまで及んでいます。2017年登場のProLiant Gen10は「ハードウェアレベルのセキュリティ」という新しい業界標準を示しました。
