用途で選ぶ - ソフトウェア・デファインド・ストレージ（SDS）

ソフトウェア・デファインド・ストレージ（SDS）選定ガイド

ソフトウェア・デファインド・ストレージ (SDS: Software Defined Storage) は、次世代のストレージ仮想化として注目を集めるソフトウェア技術です。SDSによって、x86サーバーのみで実現する共有ストレージ「ハイパーコンバージド (HCI: Hyper-Converged Infrastructure)」を構成できます。

SDSは、x86サーバーの内蔵ストレージを1つの大きな共有ストレージとして束ねるため、シンプルかつ拡張性の高いシステムが実現できます。

選定ポイント

  • 内蔵ストレージのI/O性能が重要となっています。SDSには、SSDとHDDを組み合わせたハイブリッド構成とSSDのみのオールフラッシュ構成があります。
  • 共有ストレージとして使用するため、内蔵ストレージの可用性や容量なども重視してください。
  • SDSのソフトウェアライセンスを考慮し、メニーコアの1CPUサーバーがオススメです。
