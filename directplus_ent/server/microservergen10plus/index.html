小規模システム向けタワー型サーバーとして、各拠点や部門、小規模店舗用APサーバー、中小規模ファイルサーバー用途に最適

  • W11.9 x D24.5 x H24.5cmのとてもコンパクトな筐体
  • プロセッサーはIntel® Xeon® E2224 または、Intel® Pentium® Gold G5420を1基搭載
  • 従来のGen10サーバーの2倍の演算処理を実現、VMwareをサポートし、仮想化にも対応
  • 8GB、16GBのDDR4 アンバッファ メモリ (UDIMM)を標準搭載、2スロット装備
  • 3.5型SATA HDDを最大4台で16TBを内蔵可能
  • iLO専用ポートオプションを追加することで通常のGen10サーバーと同じセキュリティ機能（iLO 5）を利用可能
  • 大型システムファンを採用することで、運転時でも最大21dBA(木の葉が触れ合う程度の音)の静音性を実現
