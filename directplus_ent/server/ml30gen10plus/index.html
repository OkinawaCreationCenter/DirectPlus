  • インテル® Xeon® E-2300プロセッサーあるいはインテル® Pentium®プロセッサーを1基搭載

  • 最大8台の2.5型ドライブ、または最大4台の3.5型ドライブを搭載可能。ホットプラグにも対応し柔軟な構成選択

  • 電源の冗長化に対応し、信頼性の向上
  • 最新OS、Windows Server 2022に対応

  • オプションのラック搭載キット利用でラックマウントにも対応(ラック搭載時5U相当)
