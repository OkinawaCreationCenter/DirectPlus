後編：サーバー選びでSQL Server 2017 移行の成否が決まる

小森博之│2016.07.21(2018.10.10アップデート)│Windows Server

「前編：SQL Server 2017 へ上手に移行するための注意点」の後編になります。

SQL Server 2008および2008 R2のサポートが、2019年7月9日（火）に終了します。一日も早い最新版のSQL Server 2017への移行をお勧めします。SQL Server 2017への移行に役立つレシピ、第2回は「安全で確実な移行のためのチェック項目」と「サーバー選びのポイント」をご紹介します。

この構成がおすすめ！

ある程度の性能を確保しつつ、コストを抑えることができる構成です。ポイントはCPU。現在最も少ないコア数のCPUを選択して、ライセンスコストを抑制します。性能が心配という場合には、同じく4コアでクロックの高いXeon Gold 5122（3.6GHz, 4コア）への変更をご検討ください。

データベース領域にフラッシュストレージを使用して、性能を大幅に向上させる“爆速構成”です。マイクロソフト社と共同で展開している「SSDアプライアンス」のローエンドモデルに相当します。大幅な性能改善を達成したお客様の実績に裏付けられたSSDアプライアンスなので、安心してご導入いただけます。
