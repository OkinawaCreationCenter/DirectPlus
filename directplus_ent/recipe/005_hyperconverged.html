HPE サーバーレシピ

ハイパーコンバージドインフラ（HCI）の使い勝手はSSDで決まる！

小川大地│2017.03.13(2019.02.01アップデート)│Windows Server

VMware社のハイパーバイザーを利用したハイパーコンバージド（HCI）は各社から様々な製品がリリースされています。HCIを実現する同社純正のソフトウェアとしてVMware vSANなども登場しているなか、Microsoft Hyper-Vについても、Windows Server 2016よりOSの標準機能だけでハイパーコンバージド環境の構築が可能になりました。
前者と違ってWindows Server 2019 Datacenterのライセンスをお持ちであれば無償で利用できるのはコスト面で大きな期待ができるでしょう。「でも、SSDの費用が掛かりそうだし、サイジングやディスク構成も難しそう」とお考えの方に、がお手頃価格で今すぐ使える、おすすめハイパーコンバージド構成をご用意いたしました。ご利用目的に合わせて2つのタイプからお選びいただけます。

さあレシピをどうぞ！

今すぐ使える、Windows Server 標準機能によるHCI

どちらのモデルにも、HCIに対応したWindows Server 2019 Datacenter エディションをバンドル。「Storage Spaces Direct（S2D）」機能を使って共有ストレージがいらないシンプルな高可用性システムを手軽に構築できます。仮想マシン数に制限がないDatacenterエディションは、仮想マシン用のWindows Serverライセンスを追加購入する必要が無いため、思い立った際にすぐに仮想マシンを構築できて便利。すぐにご利用いただけるよう、クライアント・アクセス・ライセンス（CAL）もお付けしました※。

※さらに多くのCALが必要な場合は追加購入してください。詳しくは、日本マイクロソフト「Windows Server 2019 の価格およびライセンス体系」をご参照ください。

コストダウンの鍵は、SSD

サーバーのディスクをストレージプールとして使うHCIではSSDの搭載が必須。HPE ProLiant Gen10サーバーではSSDがかなりお求めやすくなりました。これにより、予算内で性能も十分なHCIを構築できるでしょう。
今回ご用意した構成もコストパフォーマンスは抜群。特にSSDとHDDを組み合わせた「エントリーモデル」は、最小のコストで始められるハイパーコンバージドシステムとしておすすめです。

高速化ニーズにも対応

一方の「エンハンスドモデル」は、本格的な仮想化システムの要求にも耐えうるように、性能と可用性にこだわったモデルです。すべてSSDで構成するオールフラッシュ構成と、サーバーやディスクの複数同時障害にまで対応できる3面ミラー構成もサポート。加えて、本格的なシステムということで仮想マシンのセキュリティ対応ができるTPMチップも搭載しました。
